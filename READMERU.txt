Устанавливаем хак-инструменты

MalamGulita — это сборник лучших инструментов для хакинга. Прост в использовании.

Установка в Termux:
apt-get install python2 git 
apt install curl wget
 git clone https://github.com/nishinosukasa/MalamGulita.git
cd MalamGulita

Как юзать:
python2 MalamGulita.py — запуск

Всем удачного пентеста ✌️
